# HippoCamp Subsystem

The public release now uses the repository root [`README.md`](../README.md) as the
single authoritative guide.

Use the root README for:

- installation and environment setup
- RAG / search-agent reproduction
- benchmark analysis
- Docker image setup

including:

- `scripts/run_offline.py`

- `scripts/run_query.py`
- `scripts/run_evaluation.py`

For longer command cookbooks, see [`docs/reproduction.md`](../docs/reproduction.md).
